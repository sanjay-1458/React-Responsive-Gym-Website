## Getting Started

### Prerequisites

 **Node.js** (v14 or above) and **npm** installed  
  Verify with:
  ```bash
  node --version
  npm --version
```

### Clone the Repository
```bash
git clone https://github.com/sanjay-1458/React-Responsive-Gym-Website.git
cd React-Responsive-Gym-Website
```

### Install Dependencies
```
npm install
```

### Run Locally
```
npm run dev
```
